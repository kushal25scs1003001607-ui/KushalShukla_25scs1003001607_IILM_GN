# 🏋️ Personal Health Fitness Predictor  

## 📌 Overview  
This project predicts the number of calories burned using basic health data like age, weight, height, steps, and heart rate.

It uses a Machine Learning model (Linear Regression) to make predictions.

---

## 🚀 Features  
- Predicts calories burned  
- Uses Machine Learning  
- Simple and beginner-friendly  
- Fast and lightweight  

---

## 🧠 Tech Stack  
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

---

## ⚙️ Installation  

Clone the repository:
git clone https://github.com/Kushal25scs1003001607-ui/KushalShukla_25scs1003001607_IILM_GN

Go to the folder:
cd KushalShukla_25scs1003001607_IILM_GN

Install dependencies:
pip install pandas numpy scikit-learn

---

## ▶️ How to Run  

python fitness_predictor.py
